# Evals

本文档说明 evals 的目录定位。V0.1.1 不实现 Eval Runner。

## 目录

根级目录：

```text
evals/
```

模块级目录：

```text
modules/{module_name}/evals/
```

## 未来用途

evals 可用于存放：

- 输入样例。
- 预期输出说明。
- 回归检查清单。
- 手动验收材料。
- 模块级评测说明。

## 当前状态

V0.1.1 只补齐目录和说明，不新增评测执行器。

## 当前不实现

- Eval Runner。
- 自动评分。
- 批量评测。
- 外部评测平台接入。
- 生产级质量门禁。
