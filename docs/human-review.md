# Human Review

本文档只定义 Human Review 的未来边界。V0.1.1 不实现任何人审逻辑。

## 未来定位

Human Review 可用于需要人工确认、人工补充或人工驳回的运行节点。

未来可能涉及：

- review request。
- review decision。
- run pause / resume。
- review event。
- review audit。

## 当前状态

V0.1.1 仅保留文档说明，不新增 API、不新增数据模型、不改变 Run 状态机。

## 当前不实现

- 人审队列。
- 审批页面。
- run pause / resume。
- 生产级审核策略。
- 权限系统。
